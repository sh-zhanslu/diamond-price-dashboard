# Diamond Price Prediction Dashboard

Streamlit-дашборд для предсказания цены алмазов на основе характеристик: вес (carat), огранка (cut), цвет (color), чистота (clarity), глубина (depth), ширина стола (table), объём (volume).

## Модели машинного обучения (6 шт.)

| Модель | R² (на тесте) |
|--------|---------------|
| Линейная регрессия | 0.8942 |
| XGBoost (бустинг) | 0.9467 |
| CatBoost | 0.9464 |
| Random Forest (бэггинг) | 0.9386 |
| Stacking (ансамбль) | 0.9478 |
| Нейронная сеть (MLP) | 0.9093 |
